# DNS Lookup Web

A modern DNS Lookup Tool built using Python Flask.

## Features

- DNS A Record
- DNS AAAA Record
- MX Record
- NS Record
- CNAME Record
- TXT Record
- WHOIS Lookup
- Modern Responsive UI

## Technologies

- Python
- Flask
- HTML5
- CSS3
- dnspython
- python-whois

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
